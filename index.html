HTML // hypertext markup language

HTML is what forms the web. It contains text (what you see) and markup (what determines how it looks)

TAGS

tags are how a computer and a browser will read information the way we want it. Each tag means something else and is always enclosed by <>. Most of the times we must place a tag at the beginning and close it at the end of the element you want to affect.

some examples of tags are:
b = bold
em = italic
a= anchors or links
br= break
 p= paragraph

we don't need to remember all these tags, they are easily found online in websites like this one

In order to incorporate links, images or video elements into our HTML, different tags and formulas must be used. Find some examples here

some tags such as break or paragraph might have similar effects on content and text since they both separate sentences bringing them to a different level. The difference is that the first one is called inline while the last one is referred to as block.

DIFFERENCE BETWEEN INLINE AND BLOCK
inline= breaks the text into a separate line
 block= breaks the text into a sort of box that can have a specific width and height
